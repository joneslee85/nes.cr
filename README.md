## Prerequisites

* libSDL

### Installation on OSX

You can install libSDL via homebrew:

```
brew install sdl
```

## Compile

```
make
```

## Run

```
./nes rom_file.nes
```

## Controls:

* Arrows: arrows
* Z: A
* X: B
* O: Start
* P: Select
